# Stage 1 Progress Summary

## Current Stage

This repository contains the first-stage progress of my DESI sky-subtraction project.

## What Has Been Completed

### 1. Background review
I reviewed the physical origin of night-sky background and the role of atmospheric emission lines in optical spectroscopy.

### 2. Problem definition
I defined the current science goal as assessing sky-subtraction effects on DESI emission-line measurements, especially near the [OI] sky-line region around 6300 Å and 6364 Å.

### 3. DESI environment setup
I completed an initial environment check for the analysis workflow.

### 4. FITS-file inspection
I inspected DESI FITS file structure and became familiar with the main data products required for spectral analysis.

### 5. Public DESI spectrum access
I tested the retrieval and inspection of public DESI spectra.

### 6. Preliminary spectral exploration
I carried out an initial notebook-based exploration of DESI spectral content and prepared for later focused analysis near sky-line regions.

## Current Limitation
